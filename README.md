# Sistem Administrasi Warga
Proyek Tugas Akhir untuk digitalisasi administrasi lingkungan.

## Tech Stack
- **Backend:** Laravel 12 (PHP)
- **Frontend:** Vue.js 3
- **Database:** MySQL
- **Styling:** Tailwind CSS

## Fitur Utama
- Manajemen Data Warga (CRUD)
- Integrasi Data Lokasi Rumah
- Autentikasi User
